# 🚀 API de Usuários - Java Spring Boot

Projeto desenvolvido com foco em construção de APIs REST, aplicação de boas práticas de desenvolvimento e evolução contínua em Back-end com Java.

---

## 📌 Objetivo

Desenvolver uma API REST completa para gerenciamento de usuários, aplicando conceitos como:

* Arquitetura em camadas (Controller, Service, Repository)
* Operações CRUD
* Boas práticas de API REST

---

## 🛠️ Tecnologias utilizadas

* Java
* Spring Boot
* Spring Data JPA
* H2 Database
* Lombok
* Swagger (OpenAPI)

---

## ⚙️ Funcionalidades

### 🔹 CRUD de Usuários

* `GET /usuarios` → Listar todos os usuários
* `GET /usuarios/{id}` → Buscar usuário por ID
* `POST /usuarios` → Criar novo usuário
* `PUT /usuarios/{id}` → Atualizar usuário
* `DELETE /usuarios/{id}` → Remover usuário
